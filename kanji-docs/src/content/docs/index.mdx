## Features

<CardGrid stagger>
	<Card title="Modular Architecture" icon="puzzle">
		Deterministic DI container with dynamic modules. Strict boundaries, no magic.
	</Card>
	<Card title="Contract-First" icon="document">
		Define APIs with Zod schemas. Types flow end-to-end from validation to SDK.
	</Card>
	<Card title="Multi-Database" icon="database">
		PostgreSQL (Drizzle ORM) and MongoDB adapters with a common interface.
	</Card>
	<Card title="Built-in Auth" icon="seti:lock">
		JWT sessions, OAuth 2.0 (Google, GitHub, Microsoft), RBAC & ACL guards.
	</Card>
	<Card title="Real-time" icon="star">
		First-class WebSocket gateways with auth integration and contract validation.
	</Card>
	<Card title="Type-safe SDK" icon="typescript">
		Auto-generated TypeScript client from your OpenAPI spec.
	</Card>
</CardGrid>
